La méthode `addEventListener()` peut écouter une grande variété d'événements. Voici quelques exemples de types d'événements couramment utilisés :

1. **click** : Se déclenche lorsque l'utilisateur clique sur un élément⁵.
2. **mouseover** : Se déclenche lorsque l'utilisateur passe le curseur de sa souris sur un élément⁵.
3. **mouseout** : Se déclenche lorsque l'utilisateur sort son curseur d'un élément⁵.
4. **keydown** : Se déclenche lorsque l'utilisateur appuie sur une touche du clavier.
5. **keyup** : Se déclenche lorsque l'utilisateur relâche une touche du clavier.
6. **load** : Se déclenche lorsque la page a fini de charger.
7. **submit** : Se déclenche lorsque l'utilisateur soumet un formulaire.

Il existe de nombreux autres types d'événements que vous pouvez écouter avec `addEventListener()`. Pour une liste complète, vous pouvez consulter la [documentation officielle](^1^) de Mozilla Developer Network (MDN).
